使用Ubuntu16.04搭建ftp服务器

kkkkcE2024-01-062024-01-06

环境：Ubuntu16.04，内存1G，处理器1，20G硬盘，桥接

如果你不太熟练，第一次部署：

1.vsftpd

  • 没有安装，提示使用sudo apt install vsftpd命令

2.sudo apt install vsftpd


3.cat /etc/init.d/vsftpd（可选）

  • 查看vsftpd内容

4.pidof vsftpd

  • 查看正在运行名为”vsftpd”的进程，返回一或多个进程的PID

5.ps -ef | grep vsftpd（可选）

  • 显示当前系统所有进程详细信息 再把前一个命令的输出作为后一个命令的输入 查找包含vsftpd关键字的行
  • root 4156 1 0 14:11 ? 00:00:00 /usr/sbin/vsftpd /etc/vsftpd.conf 这个
  • gec 4691 2708 0 16:25 pts/18 00:00:00 grep –color=auto vsftpd

6.sudo useradd -m myftpserver1

  • root权限添加一个新用户，名为myftpserver1，-m：同时创建与用户名同名的主目录

7.sudo passwd myftpserver1

  • 给？改密码（可选）

8.cat /etc/passwd

  • 查看刚输入的密码，最下面一行是新的
  • myftpserver1:x:1001:1001::/home/myftpserver1:

9.cd /home/

-


10.在myftpserver1生成文档


11.修改vsftpd.conf

  • i:输入文本    冒号:命令    wq:写入保存退出命令    esc:退出输入文本
  • 在local_enable=YES下一行加入这2行代码
  • 找到被注释的代码chroot_local_user=YES，去掉注释，冒号，wq，退出编辑
